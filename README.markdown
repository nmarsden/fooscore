Technology
----------
* [nodejs](http://nodejs.org/)
* node modules

    * [jade](http://jade-lang.com/) - templating HTML
    * [mongoose - node MongoDB object modeling tool - http://mongoosejs.com/  
* [jQuery Mobile](http://jquerymobile.com/)
